class WebKit.CacheModel¶

Bases: GObject.GEnum

Enum values used for determining the webview cache model.

DEFAULT = 0¶

The default cache model. This is WebKit.CacheModel.WEB_BROWSER.

DOCUMENT_VIEWER = 1¶

Disable the cache completely, which substantially reduces memory usage. Useful for applications that only access a single local file, with no navigation to other pages. No remote resources will be cached.

WEB_BROWSER = 2¶

Improve document load speed substantially by caching a very large number of resources and previously viewed content.

DOCUMENT_BROWSER = 3¶

A cache model optimized for viewing a series of local files – for example, a documentation viewer or a website designer. WebKit will cache a moderate number of resources.

class WebKit.LoadStatus¶

Bases: GObject.GEnum

PROVISIONAL = 0¶

No data has been received yet, empty structures have been allocated to perform the load; the load may still fail for transport issues such as not being able to resolve a name, or connect to a port.

COMMITTED = 1¶

The first data chunk has arrived, meaning that the necessary transport requirements are stabilished, and the load is being performed.

FINISHED = 2¶

This state means that everything that was required to display the page has been loaded.

FIRST_VISUALLY_NON_EMPTY_LAYOUT = 3¶

The first layout with actual visible content happened; one or more layouts may have happened before that caused nothing to be visible on the screen, because the data available at the time was not significant enough.

FAILED = 4¶

This state means that some error occurred during the page load that prevented it from being completed. You can connect to the WebKit.WebView ::load-error signal if you want to know precisely what kind of error occurred.

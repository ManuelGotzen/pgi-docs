PackageKitPlugin.strzero(text)¶
Parameters:text (str) – The text to check
Returns:True if the string was converted correctly
Return type:bool

This function is a much safer way of doing “if (strlen (text) == 0))” as it does not rely on text being None terminated. It’s also much quicker as it only checks the first byte rather than scanning the whole string just to verify it’s not zero length.
